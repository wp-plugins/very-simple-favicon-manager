== DESCRIPTION ==
This is a very simple plugin to add an IOS, Android or Windows app icon or a favicon in the address bar of your browser.

When no favicon or app icon is added, a default favicon or app icon will be used.

Question? Please take a look at the FAQ section.

In next update I will remove the Windows app icon again, because Windows uses too many different sizes. 
And because many Android devices use the same icon as IOS devices, I will combine the two in next update. 
Next update first week of september so untill then you can clear the Android and Windows input fields.

== INSTALLATION == 
After installation upload your favicon (.ico file or .png file) and/or your IOS, Android or Windows app icon (.png file) in the media library.

Copy file link.

Go to Settings > Favicon Manager.

Paste file link and press save. 


== Frequently Asked Questions ==
= What are the best image extensions and sizes for a favicon and app icon? =
Nowadays there are several default favicon and app icon sizes, depending on browser and device. 

For desktop and laptop browsers I recommend using a square icon with extension .ico or .png and size 16x16px or 32x32px.

For IOS device I recommend using a square icon with extension .png and size 180x180px.

For Android device I recommend using a square icon with extension .png and size 192x192px.

For Windows device I recommend using a square icon with extension .png and size 270x270px.

= Can I use your plugin in combination with the new site icon feature in WordPress dashboard? =
No, you should not use both.
